# 面向数据流驱动的低延迟高通量机器学习系统

目标：不同于传统控制流驱动的计算机在数据处理与机器学习方面的低效，机器学习尤其是图像处理等任务更需要的是数据流驱动的新一代数据处理系统。本项目利用通用商用的数据流驱动系统，构建GB/秒级低延迟高通量机器学习系统。
